# Use LangSmith for agent tracing and the public trace dashboard

LangSmith is built by the LangChain team and natively integrates with LangGraph at zero instrumentation cost — two environment variables and every node execution is automatically traced. Public trace links are shareable and serve as the primary portfolio artefact demonstrating production agent discipline to hiring managers.

Langfuse was considered but adds setup overhead for LangGraph with no meaningful benefit at this stage. Can be added post-v1 to demonstrate framework-agnostic observability knowledge.
